karp - maxplus eigenvalue ( Karp algorithm )

Calling Sequence

l=karp(A)

Parameters

  • A : maxplus full or sparse matrix
  • l : eigenvalue (maxplus scalar)

Description

Maxplus eigenvalue of an irreducible matrix A. If A is not irreducible, it gives one eigenvalue but not always the largest.

Examples

karp(#(1))
karp(#([1,2;3,4]))
karp(#([1,%0;%0,2]))
karp(sparse([%0,1;3,%0]))
