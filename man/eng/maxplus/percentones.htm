Last update : 29/02/2016

percentones - (%ones) maxplus ones matrix

Calling Sequence

y=%ones(m,n)
y=%ones(x)

Description

Returns a maxplus matrix made of zeros.

returns a (m,n) matrix full of zeros. returns a matrix full of zeros with the same size as x.

Note that %ones(3) is %ones(a) with a=3 i.e. it is NOT a 3x3 matrix!
