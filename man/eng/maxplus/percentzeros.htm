percentzeros - (%zeros) maxplus zero matrix

Calling Sequence

[y]=%zeros(m,n)
[y]=%zeros(x)

Parameters

  • x,y : matrices
  • m,n : integers

Description

Maxplus matrix made of -inf (same as %0*ones).

: for an mxn matrix. : for a matrix of the same size as A. : is %zeros(a) with a=3 i.e. it is NOT a 3x3 matrix!

Examples

%zeros(3)
%zeros(3,3)
