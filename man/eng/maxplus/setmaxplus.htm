setmaxplus - maxplus constant setting

Calling Sequence

[%0,%1,%top,#]=setmaxplus()

Parameters

  • %0 : maxplus zero element (-inf)
  • %1 : maxplus one element (0)
  • %top : maxplus top element (+inf)
  • # : abbreviation for maxplus

Description

Defines the maxplus constant notations and the constructor name. It is called once when the maxplus toolbox is loaded.

Examples

[a,b,c,d]=setmaxplus();
