shift - shift maxplus linear system

Calling Sequence

[s]=shift(n,d)

Parameters

  • n : integer larger or equal to 1 which is shift in counting
  • d : real which is a shift in dating
  • s : maxplus linear system

Description

shift defines an implicit dynamic linear maxplus system associated to a shift in counting and dating. 

x(t)=u(t-n), y=d*x(t).

Examples

s=shift(2,3)
