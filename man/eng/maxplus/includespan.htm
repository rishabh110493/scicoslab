includespan - maxplus matrix image inclusion

Calling Sequence

includespan(A,B,[pre])

Parameters

  • A : full maxplus matrix
  • B : full maxplus matrix
  • pre : precision
  • b : boolean

Description

b is true if Im(B) is included in Im(A) else it is false.

Examples

a=#([1,0;0,1]);
includespan(a,#([10;10]))
includespan(a,#([2,1;1,2]))
includespan(a,#([2.0001;1]))
