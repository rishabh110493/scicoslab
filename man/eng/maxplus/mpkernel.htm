mpkernel - maxplus matrix kernel

Calling Sequence

[V,W]=mpkernel(A)

Parameters

  • A : full maxplus matrix
  • V : full maxplus matrix
  • W : full maxplus matrix

Description

Computes a weak basis of the solutions of the maxplus linear system AV=AW.

Examples

a=#([1,0;0,1]);[V,W]=mpkernel(a)
