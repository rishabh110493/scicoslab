log - natural logarithm

Calling Sequence

y=log(x)

Parameters

  • x: constant vector or constant matrix

Description

log(x) is the "element-wise" logarithm. y(i,j)=log(x(i,j)). For matrix logarithm see logm.

Examples


exp(log([1,%i,-1,-%i]))
