Lookup Tables palette

Module

Description

That palette contains various lookup table (LUT) blocks with graphical editor.

Blocks

  • INTRP2BLK_f - 2D interpolation
  • INTRPLBLK_f - Interpolation
  • LOOKUP2D - 2D Lookup table
  • LOOKUP_c - 1D Lookup table
