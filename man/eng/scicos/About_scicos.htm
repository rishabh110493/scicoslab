About Scicos...

Scicos is currently developed by the Scicos team (A. Layec, M. Najafi, F. Nassif) headed by R. Nikoukhah in the METALAU project team at INRIA, unit of Rocquencourt.

The main contributers of Scicos 4.x familly are :

  • Fady Nassif (INRIA)
    Compiler.
    Matrix/integer blocks set.
    SampleClk/Goto From/atomic blocks...

  • Masoud Najafi (INRIA)
    Modelica implementation.
    Simulator.

  • Ramine Nikoukhah (INRIA)
    Compiler algorithm.
    Multi-windows graphical editor behavior.

  • Alan Layec (INRIA)
    ScicosLab interfacing functions.
    C code generator.
    HTML documentation.
    Themed Tcl/Tk widgets design.

  • Simone Mannori/Serge Steer (INRIA)
    Graphical editor.

  • Benoit Bayol
    New graphic scope blocks set.
