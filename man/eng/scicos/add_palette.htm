add_palette - Add Scicos palette

Module

  • Scicos

Description

Add a palette to the end of the list of the scicos palette. If the palette exists with same title and same path noting is done. If a palette with same name but different path exist then the palette is added with a suffixe (x) Note that scicos_pal variable is locally recreated and passed to the upper environnement via scicos_pal=resume(scicos_pal)

Calling Sequence 

add_palette(Title,path)

Parameters

  • Title
    The name of the palette.
  • path
    The path of the palette.
