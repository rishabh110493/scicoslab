coserror - Abort the simulation and display an error message

Module

  • Scicos

Description

Function to set a specific error message during the simulation for the current block. If it is used, then after the execution of the computational function of the block, the simulator will end and will return the error message specified in argument.

Calling Sequence 

Coserror(errmsg)

Parameters

  • errmsg : a string that gives the error message to be displayed for the user

See Also

  • set_blockerror - Set the block error number (Scilab Function)
