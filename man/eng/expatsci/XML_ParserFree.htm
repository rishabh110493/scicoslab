XML_ParserFree - Free a XML pointer

Calling Sequence

XML_ParserFree()

Parameters

  • %ptr : a Scilab pointer pointing to an Expat object.

Description

This function free a XML pointer previously created with XML_ParserCreate.

Examples

  XML_ParserFree(%ptr);
