startup - startup file

Description

The startup file SCIHOME/.scilab and SCIHOME/scilab.ini in are automatically executed (if present) when ScicosLab is invoked, in addition with the file scilab.star in the ScicosLab directory (SCI).

Remarks

Last line of startup file must be terminated by a newline to be taken into account.

SCIHOME definition : On Windows : C:/Documents and Settings/<User>/ScicosLab/<ScicosLab-Version> On Linux/Unix : /home/<User>/.ScicosLab/<ScicosLab-Version>
