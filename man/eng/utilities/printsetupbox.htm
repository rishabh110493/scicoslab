printsetupbox - Display print dialog box (Windows)

Calling Sequence

printsetupbox()
bOK=printsetupbox()

Description

bOK=printsetupbox() displays the built-in printing dialogbox for Windows and configure the printer. returns %T if you push 'OK' button
