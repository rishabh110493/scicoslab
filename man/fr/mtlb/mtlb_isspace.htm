mtlb_isspace - Fonction d'émulation de la fonction Matlab isspace

Description

Il n'y a pas de fonction Scilab équivalente à la fonction Matlab isspace mais son comportement peut être reproduit.

La fonction mtlb_isspace(A) est utilisée par mfile2sci pour remplacer isspace(A) quand il n'était pas possible de savoir ce qu'étaient les paramètres d'entrée pendant la conversion d'un code Matlab vers Scilab. Cette fonction va déterminer la sémantique correcte pendant l'exécution. Pour obtenir un code plus performant on peut remplacer les appels à mtlb_isspace :

  • Si A est une matrice de chaînes de caractères mtlb_isspace(A) peut être remplacé par asciimat(A)==32
  • Si A n'est pas une matrice de chaînes de caractères mtlb_isspace(A) peut être remplacé par zeros(A)

    • Attention : mtlb_isspace ne doit pas être utilisée pour une programmation manuelle.
