mtlb_fftshift - Fonction d'émulation de la fonction Matlab fftshift

Description

Dans plusieurs cas particuliers, la fonction fftshift se comporte différemment dans Matlab et dans Scilab :

  • Avec les chaînes de caractère : étant donné que les chaînes de caractères sont vues différemment par Matlab et Scilab, le résulat peut être différent pour ce type d'entrée.
  • Avec deux paramètres d'entrée : Matlab autorise dim a être supérieur au nombre de dimension de A mais pas Scilab (message d'erreur).

    • La fonction mtlb_fftshift(A,[dim]) est utilisée par mfile2sci pour remplacer fftshift(A,[dim]) quand il n'était pas possible de savoir ce qu'étaient les paramètres d'entrée pendant la conversion d'un code Matlab vers Scilab. Cette fonction va déterminer la sémantique correcte pendant l'exécution. Pour obtenir un code plus performant on peut remplacer les appels à mtlb_fftshift :

  • Si A n'est pas une matrice de chaînes de caractères, mtlb_fftshift(A) peut être remplacé par fftshift(A)
  • Si A n'est pas une matrice de chaînes de caractères et dim est inférieur ou égal au nombre de dimension de A, mtlb_fftshift(A,dim) peut être remplacé par fftshift(A,dim)

    • Attention : mtlb_fftshift ne doit pas être utilisée pour une programmation manuelle.
