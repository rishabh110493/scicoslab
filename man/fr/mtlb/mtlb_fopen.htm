mtlb_fopen - Fonction d'émulation de la fonction Matlab fopen

Description

Dans plusieurs cas particuliers, la fonction Matlab fopen et la fonction Scilab mopen se comportent différemment :

  • La fonction Scilab ne renvoie pas de pessage d'erreur utilisable
  • L'indentifieur de fichier n'est pas initialisé dans Scilab en cas d'erreur alors que Matlab lui fixe la valeur -1
  • La fonction Matlab peut fonctionner avec des arguments qui n'existent pas en Scilab : options de permission...

    • La fonction mtlb_fopen(filename,permission) est utilisée par mfile2sci pour remplacer fopen(filename,permission) quand il n'était pas possible de savoir ce qu'étaient les paramètres d'entrée pendant la conversion d'un code Matlab vers Scilab. Cette fonction va déterminer la sémantique correcte pendant l'exécution. Pour obtenir un code plus performant on peut remplacer les appels à mtlb_fopen :

  • Si le message d'erreur n'est pas utilisé et si aucune erreur ne peut avoir lieu, mtlb_fopen(filename,permission) peut être remplacé par mopen(filename,permission,0)

    • Attention : mtlb_fopen ne doit pas être utilisée pour une programmation manuelle.
