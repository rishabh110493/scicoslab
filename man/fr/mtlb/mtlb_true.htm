mtlb_true - Fonction d'émulation de la fonction Matlab true

Description

Il n'y a pas d'équivalent Scilab pour la fonction Matlab true, cependant la fonction Scilab ones retourne un résultat interprété de la même façon pour Scilab.

Dans plusieurs cas particuliers, la fonction Matlab true et la fonction Scilab onesse comporte différemment :

  • Avec une entrée qui est un scalaire : La fonction Matlab true retourne une matrice n x n alors que la fonction Scilab ones retourne 1.

    • La fonction mtlb_true(A) est utilisée par mfile2sci pour remplacer true(A) quand il n'était pas possible de savoir ce qu'étaient les opérandes pendant la conversion d'un code Matlab vers Scilab. Cette fonction va déterminer la sémantique correcte pendant l'exécution. Pour obtenir un code plus performant on peut remplacer les appels à mtlb_true :

  • Si A est un scalaire mtlb_true(A) peut être remplacé par ones(A,A)
  • Si A n'est pas un scalaire mtlb_true(A) peut être remplacé par ones(A)

    • Attention : mtlb_true ne doit pas être utilisée pour une programmation manuelle.
