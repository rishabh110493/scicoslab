mtlb_grid - Fonction d'émulation de la fonction Matlab grid

Description

Il n'y a pas de fonction Scilab equivalente à la fonction Matlab grid mais il existe des instructions équivalentes

La fonction mtlb_grid(flag_or_handle[,flag]) est utilisée par mfile2sci pour remplacer grid(flag_or_handle[,flag]) quand il n'était pas possible de savoir ce qu'étaient les paramètres d'entrée pendant la conversion d'un code Matlab vers Scilab. Cette fonction va déterminer la sémantique correcte pendant l'exécution. Pour obtenir un code plus performant on peut remplacer les appels à mtlb_grid :

  • Avec un paramètre d'entrée, si flag est égal à "on" mtlb_grid(flag) peut être remplacé par set(gca(),"grid",[1 1])
  • Avec un paramètre d'entrée, si flag est égal à "off" mtlb_grid(flag) peut être remplacé par set(gca(),"grid",[-1 -1])
  • Avec deux paramètres d'entrée, si flag est égal à "on" mtlb_grid(axes_handle,flag) peut être remplacé par axes_handle.grid=[1 1]
  • Avec un paramètre d'entrée, si flag est égal à "off" mtlb_grid(axes_handle,flag) peut être remplacé par axes_handle.grid=[-1 -1]

    • Attention : mtlb_grid ne doit pas être utilisée pour une programmation manuelle.
