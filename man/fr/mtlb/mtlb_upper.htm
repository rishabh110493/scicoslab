mtlb_upper - Fonction d'émulation de la fonction Matlab upper

Description

Dans plusieurs cas particuliers, la fonction Matlab upper(A) et la fonction Scilab convstr(A,"u") se comportent différemment :

  • Si A n'est pas une chaîne de caractères : la fonction Matlab upper peut être utilisée avec A n'étant pas une chaîne de caractères mais pas la fonction Scilab convstr.

    • La fonction mtlb_upper(A) est utilisée par mfile2sci pour remplacer upper(A) quand il n'était pas possible de savoir ce qu'étaient les paramètres d'entrée pendant la conversion d'un code Matlab vers Scilab. Cette fonction va déterminer la sémantique correcte pendant l'exécution. Pour obtenir un code plus performant on peut remplacer les appels à mtlb_upper :

  • Si A est une matrice de chaînes de caractères mtlb_upper(A) peut être remplacé par convstr(A,"u")
  • Si A n'est pas une matrice de chaînes de caractères mtlb_upper(A) peut être remplacé par A

    • Attention : mtlb_upper ne doit pas être utilisée pour une programmation manuelle.
