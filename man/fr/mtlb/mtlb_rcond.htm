mtlb_rcond - Fonction d'émulation de la fonction Matlab rcond

Description

Dans plusieurs cas particuliers, la fonction rcond se comporte différemment dans Matlab et dans Scilab :

  • Avec une matrice vide : La fonction Matlab rcond retourne Inf et Scilab retourne []

    • La fonction mtlb_rcond(A) est utilisée par mfile2sci pour remplacer rcond(A) quand il n'était pas possible de savoir ce qu'était le paramètre d'entrée pendant la conversion d'un code Matlab vers Scilab. Cette fonction va déterminer la sémantique correcte pendant l'exécution. Pour obtenir un code plus performant on peut remplacer les appels à mtlb_rcond :

  • Si A n'est pas une matrice vide mtlb_rcond(A) peut être remplacé par rcond(A)

    • Attention : mtlb_rcond ne doit pas être utilisée pour une programmation manuelle.
