mtlb_find - Fonction d'émulation de la fonction Matlab find

Description

Dans plusieurs cas particuliers, la fonction find se comporte différemment dans Matlab et dans Scilab :

  • Avec un paramètre d'entrée qui est un vecteur colonne ou une matrice : La fonction Matlab find retourne un(des) vecteur(s) colonne alors que Scilab retourne un(des) vecteur(s) ligne.
  • Avec trois paramètres de sortie : La fonction Matlab find peut retourner trois valeurs mais pas la fonction Scilab.

    • La fonction [i[,j[,v]]] = mtlb_find(B) est utilisée par mfile2sci pour remplacer [i[,j[,v]]] = find(B) quand il n'était pas possible de savoir ce qu'étaient les paramètres d'entrée pendant la conversion d'un code Matlab vers Scilab. Cette fonction va déterminer la sémantique correcte pendant l'exécution. Pour obtenir un code plus performant on peut remplacer les appels à mtlb_find :

  • Pour un appel avec un seul paramètre de sortie, si B est un vecteur ligne, i = mtlb_find(B) peut être remplacé par i = find(B)
  • Pour un appel avec un seul paramètre de sortie, si B n'est pas un vecteur ligne, i = mtlb_find(B) peut être remplacé par i = matrix(find(B),-1,1)
  • Pour un appel avec deux paramètres de sortie, si B est un vecteur ligne, [i,j] = mtlb_find(B) peut être remplacé par [i,j] = find(B)
  • Pour un appel avec un seul paramètre de sortie, si B n'est pas un vecteur ligne, [i,j] = mtlb_find(B) peut être remplacé par [i,j] = find(B);i = matrix(i,-1,1);j = matrix(j,-1,1);

    • Attention : mtlb_find ne doit pas être utilisée pour une programmation manuelle.
