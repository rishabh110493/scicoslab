mtlb_sort - Fonction d'émulation de la fonction Matlab sort

Calling Sequence

P = mtlb_sort(X)
P = mtlb_sort(X,dim[,mode])

Parameters

  • X: un scalaire, un vecteur, une matrice ou hypermatrice d'elements compris dans [-1 1]un scalaire, un vecteur ou une matrice de réels, de booléens ou une string
  • ndim: un entier positif
  • mode:une chaine de caracteres ("ascend" or "descend")

Description

Dans plusieurs cas particuliers, la fonction sort de Matlab et la fonction gsort de Scilab se comportent différemment :

  • Si X est un vecteur alors l'appel de la fonction sort(X) de Matlab est équivalent à  l'appel de la fonction sort(X,'g','i') de Scilab.
  • dim=1 (resp. 2) de Matlab est equivalent au flag 'r' (resp. 'c') de Scilab.
  • mode="ascend" (resp. "descend") est équivalent au flag 'i' (resp 'd') de Scilab.

    • La fonction mtlb_sort(X,dim[,mode]) est utilisée par mfile2sci pour remplacer sort(X,dim[,mode]) quand il n'était pas possible de savoir ce qu'étaient les paramètres d'entrée pendant la conversion d'un code Matlab vers Scilab. Cette fonction va déterminer la sémantique correcte pendant l'exécution. Pour obtenir un code plus performant on peut remplacer les appels à  mtlb_sort par des appels gsort.

    Attention : mtlb_sort ne doit pas être utilisée pour une programmation manuelle.
