mtlb_strcmp - Fonction d'émulation de la fonction Matlab strcmp

Description

Il n'y a pas dans Scilab de fonction équivalente à la fonction Matlab strcmp mais il existe des instructions équivalentes.

La fonction mtlb_strcmp(A,B) est utilisée par mfile2sci pour remplacer strcmp(A,B) quand il n'était pas possible de savoir ce qu'étaient les paramètres d'entrée pendant la conversion d'un code Matlab vers Scilab. Cette fonction va déterminer la sémantique correcte pendant l'exécution. Pour obtenir un code plus performant on peut remplacer les appels à mtlb_strcmp :

  • Si A et B sont des chaînes de caractères mtlb_strcmp(A,B) peut être remplacé par A==B
  • Si A et/ou B n'est pas une chaîne de caractères mtlb_strcmp(A,B) peut être remplacé par 0

    • Attention : mtlb_strcmp ne doit pas être utilisée pour une programmation manuelle.
