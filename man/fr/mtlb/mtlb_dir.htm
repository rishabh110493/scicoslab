mtlb_dir - Fonction d'émulation de la fonction Matlab dir

Description

La fonction dir se comporte différemment dans Matlab et dans Scilab :

  • Quand le résultat est stocké dans une variable: La fonction Matlab dir renvoie une structure mais la fonction Scilab retourne une tlist de type 'dir', ce qui fait que les données ne peuvent être extraites de la même manière.

    • La fonction mtlb_dir([path]) est utilisée par mfile2sci pour remplacer dir([path]) quand la sortie est stockée dans une variable. Il n'est pas possible de remplacer cette fonction, sinon (si on la remplace par dir) les données ne pourront pas être extraites comme dans Matlab. Par exemple, l'équivalent Scilab pour l'instruction Matlab L=dir;file_name=L(1).name; est L=dir();file_name=L.name(1);.

    Attention : mtlb_dir ne doit pas être utilisée pour une programmation manuelle.
