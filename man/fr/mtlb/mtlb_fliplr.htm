mtlb_fliplr - Fonction d'émulation de la fonction Matlab fliplr

Description

Dans plusieurs cas particuliers, la fonction fliplr se comporte différemment dans Matlab et dans Scilab :

  • Avec un paramètre d'entrée de type chaîne de caractère : étant donné que Scilab et Matlab ne considèrent pas les chaînes de caractère de la même façon, le résultat pout être différent pour une entrée de ce type.

    • La fonction mtlb_fliplr(A) est utilisée par mfile2sci pour remplacer fliplr(A) quand il n'était pas possible de savoir ce qu'étaient les paramètres d'entrée pendant la conversion d'un code Matlab vers Scilab. Cette fonction va déterminer la sémantique correcte pendant l'exécution. Pour obtenir un code plus performant on peut remplacer les appels à mtlb_fliplr :

  • Si A n'est pas une matrice de chaînes de caractères, mtlb_fliplr(A) peut être remplacé par fliplr(A)

    • Attention : mtlb_fliplr ne doit pas être utilisée pour une programmation manuelle.
