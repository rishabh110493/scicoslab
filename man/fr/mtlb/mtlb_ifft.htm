mtlb_fft - Fonction d'émulation de la fonction Matlab ifft

Description

Dans plusieurs cas particuliers, la fonction Matlab ifft et la fonction Scilab fft se comportent différemment :

  • Avec un paramètre d'entrée : si ce paramètre est un scalaire ou un vecteur l'équivalent Scilab pour la fonction Matlab ifft(A) est fft(A,1) et si ce paramètre est une matrice l'équivalent Scilab pour l'appel Matlab à ifft est fft(A,1,2,1)

    • La fonction mtlb_ifft(X[,n,[job]]) est utilisée par mfile2sci pour remplacer ifft(X[,n,[job]]) quand il n'était pas possible de savoir ce qu'étaient les entrées pendant la conversion d'un code Matlab vers Scilab. Cette fonction va déterminer la sémantique correcte pendant l'exécution. Pour obtenir un code plus performant on peut remplacer les appels à mtlb_ifft :

  • Si X est un scalaire ou un vecteur mtlb_ifft(X,1) peut être remplacé par fft(X,1)
  • Si X est une matrice mtlb_ifft(X,1) peut être remplacé par fft(X,1,2,1)

    • Attention : mtlb_ifft ne doit pas être utilisée pour une programmation manuelle.
