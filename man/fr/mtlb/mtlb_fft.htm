mtlb_fft - Fonction d'émulation de la fonction Matlab fft

Description

Dans plusieurs cas particuliers, la fonction fft se comporte différemment dans Matlab et dans Scilab :

  • Avec un paramètre d'entrée : si ce paramètre est un scalaire ou un vecteur l'équivalent Scilab pour la fonction Matlab fft est fft(...,-1) et si ce paramètre est une matrice l'équivalent Scilab pour l'appel Matlab à fft est fft(...,-1,2,1)

    • La fonction mtlb_fft(X[,n,[job]]) est utilisée par mfile2sci pour remplacer fft(X[,n,[job]]) quand il n'était pas possible de savoir ce qu'étaient les entrées pendant la conversion d'un code Matlab vers Scilab. Cette fonction va déterminer la sémantique correcte pendant l'exécution. Pour obtenir un code plus performant on peut remplacer les appels à mtlb_fft :

  • Si X est un scalaire ou un vecteur mtlb_fft(X,-1) peut être remplacé par fft(X,-1)
  • Si X est une matrice mtlb_fft(X,-1) peut être remplacé par fft(X,-1,2,1)

    • Attention : mtlb_fft ne doit pas être utilisée pour une programmation manuelle.
