mtlb_close - Fonction d'émulation de la fonction Matlab close

Description

Suivant le type de la figure courante (uicontrol ou graphique), l'équivalent Scilab pour la fonction Matlab close est différent :

  • Si la figure courante est de type uicontrol : l'équivalent Scilab pour la fonction Matlab close est close
  • Si la fenêtre courante est une fenêtre graphique : l'équivalent Scilab pour la fonction Matlab close est xdel ou delete
  • Dans Scilab on ne récupère pas de statut d'erreur.

    • La fonction mtlb_close([h]) est utilisée par mfile2sci pour remplacer close([h]) quand il n'était pas possible de savoir ce qu'étaient les paramètres d'entrée pendant la conversion d'un code Matlab vers Scilab. Cette fonction va déterminer la sémantique correcte pendant l'exécution. Pour obtenir un code plus performant on peut remplacer les appels à mtlb_close :

  • Si h est une figure de type uicontrol, mtlb_close(h) peut être remplacé par close(h)
  • Si h est une figure graphique, mtlb_close(h) peut être remplacé par delete(h)

    • Attention : mtlb_close ne doit pas être utilisée pour une programmation manuelle.
