mtlb_legendre - Fonction d'émulation de la fonction Matlab legendre

Calling Sequence

P=mtlb_legendre(n,X)
P=mtlb_legendre(n,X[,normflag])

Parameters

  • X: un scalaire, un vecteur, une matrice ou hypermatrice d'elements compris dans [-1 1]
  • n: un entier positif
  • normflag: une chaine de caracteres ('sch' ou 'norm')

Description

Dans plusieurs cas particuliers, la fonction legendre se comporte différemment dans Matlab et dans Scilab :

  • L'appel de fonction Scilab legendre(m,n,X) évalue la fonction de legendre de degré n et d'ordre m pour tous les éléments de X. Par contre l'appel de fonction Scilab legendre(m,n,X) évalue la fonction de legendre de degré n et d'ordre m=0,1,...,n pour tous les éléments de X.
  • Le mode normflag='sch' n'est pas implémenté avec Scilab
  • La fonction legendre de Scilab ne fonctionne pas si X est une hypermatrice

    • La fonction mtlb_legendre(n,X[,normflag]) est utilisée par mfile2sci pour remplacer legendre(n,X[,normflag]) quand il n'était pas possible de savoir ce qu'étaient les paramètres d'entrée pendant la conversion d'un code Matlab vers Scilab. Cette fonction va déterminer la sémantique correcte pendant l'exécution. Pour obtenir un code plus performant on peut remplacer les appels à  mtlb_legendre :

  • If X is a scalar, a vector or a matrix mtlb_legendre(n,X[,'norm']) may be replaced by legendre(n,0:n,X[,'norm'])

    • Attention : mtlb_legendre ne doit pas être utilisée pour une programmation manuelle.
