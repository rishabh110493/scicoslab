mtlb_full - Fonction d'émulation de la fonction Matlab full

Description

Dans plusieurs cas particuliers, la fonction full se comporte différemment dans Matlab et dans Scilab :

  • Avec un paramètre d'entrée de type chaîne de caractère : La fonction Matlab full peut être utilisée avec des chaînes de caractères ce qui n'est pas possible dans Scilab.
  • Avec un paramètre d'entrée de type booléen : La fonction Matlab full peut être utilisée avec des booléens ce qui n'est pas possible dans Scilab.

    • La fonction mtlb_full(A) est utilisée par mfile2sci pour remplacer full(A) quand il n'était pas possible de savoir ce qu'étaient les paramètres d'entrée pendant la conversion d'un code Matlab vers Scilab. Cette fonction va déterminer la sémantique correcte pendant l'exécution. Pour obtenir un code plus performant on peut remplacer les appels à mtlb_full :

  • Si A est une matrice de doubles mtlb_full(A) peut être remplacé par full(A)
  • Si A est une matrice de booléens mtlb_full(A) peut être remplacé par full(bool2s(A))

    • Attention : mtlb_full ne doit pas être utilisée pour une programmation manuelle.
