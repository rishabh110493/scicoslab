mtlb_strrep - Fonction d'émulation de la fonction Matlab strrep

Description

Dans plusieurs cas particuliers, la fonction Matlab strrep et la fonction Scilab strsubst se comportent différemment :

  • Avec des paramètres d'entrée n'étant pas des chaînes de caractère : La fonction Matlab strrep peut être utilisée avec des entrées n'étant pas des chaînes de caractères mais pas la fonction Scilab strsubst.

    • La fonction mtlb_strrep(A,B,C) est utilisée par mfile2sci pour remplacer strrep(A,B,C) quand il n'était pas possible de savoir ce qu'étaient les paramètres d'entrée pendant la conversion d'un code Matlab vers Scilab. Cette fonction va déterminer la sémantique correcte pendant l'exécution. Pour obtenir un code plus performant on peut remplacer les appels à mtlb_strrep :

  • Si A, B et C sont des chaînes de caractères y = mtlb_strrep(A,B,C) peut être remplacé par strsubst(A,B,C)

    • Attention : mtlb_strrep ne doit pas être utilisée pour une programmation manuelle.
