mtlb_ones - Fonction d'émulation de la fonction Matlab ones

Description

Dans plusieurs cas particuliers, la fonction ones se comporte différemment dans Matlab et dans Scilab :

  • Avec une entrée qui est un scalaire : La fonction Matlab ones retourne une matrice n x n alors que Scilab retourne 1.

    • La fonction mtlb_ones(A) est utilisée par mfile2sci pour remplacer ones(A) quand il n'était pas possible de savoir ce qu'étaient les opérandes pendant la conversion d'un code Matlab vers Scilab. Cette fonction va déterminer la sémantique correcte pendant l'exécution. Pour obtenir un code plus performant on peut remplacer les appels à mtlb_ones :

  • Si A est un scalaire mtlb_ones(A) peut être remplacé par ones(A,A)
  • Si A n'est pas un scalaire mtlb_ones(A) peut être remplacé par ones(A)

    • Attention : mtlb_ones ne doit pas être utilisée pour une programmation manuelle.
