mtlb_uint32 - Fonction d'émulation de la fonction Matlab uint32

Description

Dans plusieurs cas particuliers, la fonction uint32 se comporte différemment dans Matlab et dans Scilab :

  • Avec un paramètre d'entrée complexe : La fonction Matlab uint32 peut être utilisée avec des complexes mais pas la fonction Scilab.
  • Avec la valeur %inf : Matlab retourne 4294967295 alors que Scilab retourne 0.

    • La fonction mtlb_uint32(A) est utilisée par mfile2sci pour remplacer uint32(A) quand il n'était pas possible de savoir ce qu'était le paramètre d'entrée pendant la conversion d'un code Matlab vers Scilab. Cette fonction va déterminer la sémantique correcte pendant l'exécution. Pour obtenir un code plus performant on peut remplacer les appels à mtlb_uint32 :

  • Si A ne contient pas de valeur %inf, %nan ou -%nanmtlb_uint32(A) peut être remplacé par uint32(A)

    • Attention : mtlb_uint32 ne doit pas être utilisée pour une programmation manuelle.
