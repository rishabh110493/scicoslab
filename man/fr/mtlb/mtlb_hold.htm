mtlb_hold - Fonction d'émulation de la fonction Matlab hold

Description

Il n'y a pas de fonction Scilab equivalente à la fonction Matlab hold mais il existe des instructions équivalentes

La fonction mtlb_hold(flag) est utilisée par mfile2sci pour remplacer hold flag quand il n'était pas possible de savoir ce qu'étaient les paramètres d'entrée pendant la conversion d'un code Matlab vers Scilab. Cette fonction va déterminer la sémantique correcte pendant l'exécution. Pour obtenir un code plus performant on peut remplacer les appels à mtlb_hold :

  • Si flag est égal à "on" mtlb_hold(flag) peut être remplacé par set(gca(),"auto_clear","off")
  • Si flag est égal à "off" mtlb_hold(flag) peut être remplacé par set(gca(),"auto_clear","on")

    • Attention : mtlb_hold ne doit pas être utilisée pour une programmation manuelle.
