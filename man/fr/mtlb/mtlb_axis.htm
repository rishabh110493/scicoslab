mtlb_axis - Fonction d'émulation de la fonction Matlab axis

Calling Sequence

mtlb_axis(X)
mtlb_axis(st)
mtlb_axis(axeshandle, ...)
[mode,visibility,direction]=mtlb_axis('state')

Parameters

  • X: un vecteur de réels ([xmin xmax ymin ymax] ou [xmin xmax ymin ymax zmin zmax])
  • st: une chaine de caractères ('auto', 'manual', 'tight', 'ij', 'xy', 'equal', 'square', 'vis3d', 'off', 'on')
  • axeshandle:handle sur l'axe courant

Description

La fonction Matlab axis n'a pas de fonction Scilab équivalente :

La fonction mtlb_axis(n,X[,normflag]) est utilisée par mfile2sci pour remplacer axis(...) quand il n'était pas possible de savoir ce qu'étaient les paramètres d'entrée pendant la conversion d'un code Matlab vers Scilab. Cette fonction va déterminer la sémantique correcte pendant l'exécution. ('fill' , 'image' et 'normal' ne sont pas implémentés).Pour obtenir un code plus performant on peut remplacer les appels à  mtlb_axis par les appels à  get(axeshandle,prop), ou prop est une propriété des axes (see axis_properties)

Attention : mtlb_axis ne doit pas être utilisée pour une programmation manuelle.
