mtlb_isletter - Fonction d'émulation de la fonction Matlab isletter

Description

Il n'y a pas de fonction Scilab équivalente à la fonction Matlab isletter et les expressions équivalentes sont assez compliquées, donc mfile2sci utilise mtlb_isletter(A)pour remplacer isletter(A). Pour obtenir un code plus performant on peut remplacer les appels à mtlb_isletter :

  • Si A n'est pas une chaîne de caractères tf = mtlb_isletter(A) peut être remplacé par tf = zeros(A)
  • Si A est une chaîne de caractères tf = mtlb_isletter(A) peut être remplacé par tf = (asciimat(A)>=65&asciimat(A)<=90)|(asciimat(A)>=97&asciimat(A)<=122)

    • Attention : mtlb_isletter ne doit pas être utilisée pour une programmation manuelle.
