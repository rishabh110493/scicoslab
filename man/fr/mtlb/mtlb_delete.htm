mtlb_delete - Fonction d'émulation de la fonction Matlab delete

Description

La fonction mtlb_delete(A) est utilisée par mfile2sci pour remplacer delete(A) quand il n'était pas possible de savoir ce qu'était le paramètres d'entrée pendant la conversion d'un code Matlab vers Scilab. Cette fonction va déterminer la sémantique correcte pendant l'exécution. Pour obtenir un code plus performant on peut remplacer les appels à mtlb_delete :

  • Si A est une chaîne de caractères mtlb_delete(A) peut être remplacé par mdelete(A)
  • Si A est un handle graphique mtlb_delete(A) peut être remplacé par delete(A)

    • Attention : mtlb_delete ne doit pas être utilisée pour une programmation manuelle.
