mtlb_strcmpi - Fonction d'émulation de la fonction Matlab strcmpi

Description

Il n'y a pas dans Scilab de fonction équivalente à la fonction Matlab strcmpi mais il existe des instructions équivalentes.

La fonction mtlb_strcmpi(A,B) est utilisée par mfile2sci pour remplacer strcmpi(A,B) quand il n'était pas possible de savoir ce qu'étaient les paramètres d'entrée pendant la conversion d'un code Matlab vers Scilab. Cette fonction va déterminer la sémantique correcte pendant l'exécution. Pour obtenir un code plus performant on peut remplacer les appels à mtlb_strcmpi :

  • Si A et B sont des chaînes de caractères mtlb_strcmpi(A,B) peut être remplacé par convstr(A)==convstr(B)
  • Si A et/ou B n'est pas une chaîne de caractères mtlb_strcmpi(A,B) peut être remplacé par 0

    • Attention : mtlb_strcmpi ne doit pas être utilisée pour une programmation manuelle.
