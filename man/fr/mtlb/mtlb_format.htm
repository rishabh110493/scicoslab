mtlb_format - Fonction d'émulation de la fonction Matlab format

Description

Dans plusieurs cas particuliers, la fonction format se comporte différemment dans Matlab et dans Scilab :

  • Certains formats Matlab n'existent pas en Scilab.
  • La séquence d'appel de format est différente en Scilab et Matlab.

    • La fonction mtlb_format(type,prec) est utilisée par mfile2sci pour remplacer format([type prec]) quand il n'était pas possible de savoir ce qu'étaient les paramètres d'entrée pendant la conversion d'un code Matlab vers Scilab. Cette fonction va déterminer la sémantique correcte pendant l'exécution. Pour obtenir un code plus performant on peut remplacer les appels à mtlb_format :

  • Si type="" et prec=""mtlb_format("","") peut être remplacé par format("v",6)
  • Si type="+" et prec=""mtlb_format("+","") peut être remplacé par format(6)
  • Si type="long" et prec=""mtlb_format("long","") peut être remplacé par format(16)
  • Si type="long" et prec="e"mtlb_format("long","e") peut être remplacé par format("e"16)
  • Si type="long" et prec="g"mtlb_format("long","g") peut être remplacé par format("e"16)
  • Si type="short" et prec=""mtlb_format("short","") peut être remplacé par format(6)
  • Si type="short" et prec="e"mtlb_format("short","e") peut être remplacé par format("e"6)
  • Si type="short" et prec="g"mtlb_format("short","g") peut être remplacé par format("e"6)

    • Attention : mtlb_format ne doit pas être utilisée pour une programmation manuelle.
