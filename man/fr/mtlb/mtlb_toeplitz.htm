mtlb_toeplitz - Fonction d'émulation de la fonction Matlab toeplitz

Description

Dans plusieurs cas particuliers, la fonction toeplitz se comporte différemment dans Matlab et dans Scilab :

  • Avec un paramètre d'entrée : si celui-ci est complexe ou si c'est une matrice, la valeur de sortie des fonctions toeplitz de Matlab et Scilab sera différente.
  • Avec deux paramètres d'entrée : si ce sont des vecteurs et que leurs premiers éléments sont différents, Scilab renvoie un message d'erreur alors que Matlab donne priorité à l'élément colonne. Si ce sont des matrices, la valeur retournée par la fonction toeplitz sera différente dans Matlab et dans Scilab.

    • La fonction mtlb_toeplitz(c[,r]) est utilisée par mfile2sci pour remplacer toeplitz(c[,r]) quand il n'était pas possible de savoir ce qu'étaient les paramètres d'entrée pendant la conversion d'un code Matlab vers Scilab. Cette fonction va déterminer la sémantique correcte pendant l'exécution. Pour obtenir un code plus performant on peut remplacer les appels à mtlb_toeplitz :

  • Pour les appels avec un paramètre d'entrée, si c est un scalaire ou un vecteur réel, mtlb_toeplitz(c) peut être remplacé par toeplitz(c)
  • Pour les appels avec deux paramètres d'entrée, si c et r sont des scalaires ou des vecteurs et que leurs premiers éléments sont égaux, mtlb_toeplitz(c,r) peut être remplacé par toeplitz(c,r)

    • Attention : mtlb_toeplitz ne doit pas être utilisée pour une programmation manuelle.
