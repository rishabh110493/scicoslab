mtlb_plot - Fonction d'émulation de la fonction Matlab plot

Description

La fonction Matlab plot et la fonction Scilab plot2d se comportent différemment.

La fonction mtlb_plot(...) est utilisée par mfile2sci pour remplacer plot(...) quand il n'était pas possible de savoir ce qu'étaient les paramètres d'entrée pendant la conversion d'un code Matlab vers Scilab. Cette fonction va déterminer la sémantique correcte pendant l'exécution. Pour obtenir un code plus performant on peut remplacer les appels à mtlb_plot quand il n'y a pas de valeur de sortie, dans ce cas, les couleurs doivent être changées par l'utilisateur:

  • Avec un paramètre d'entrée, si Y est réel mtlb_plot(Y) peut être remplacé par plot2d(Y)
  • Avec un paramètre d'entrée, si Y est complex mtlb_plot(Y) peut être remplacé par plot2d(real(Y),imag(Y))
  • Avec deux paramètres d'entrée X et Y, si Y n'est pas une chaîne de caractères, mtlb_plot(X,Y) peut être remplacé par plot2d(X,Y)

    • Attention : mtlb_plot ne doit pas être utilisée pour une programmation manuelle.
