mtlb_diag - Fonction d'émulation de la fonction Matlab diag

Description

Dans plusieurs cas particuliers, la fonction diag se comporte différemment dans Matlab et dans Scilab :

  • Avec des matrices de chaînes de caractères : La fonction diag de Scilab considère les chaînes comme des objets alors que Matlab considère chaque caractère séparément.

    • La fonction y = mtlb_diag(x[,dim]) est utilisée par mfile2sci pour remplacer y = diag(x[,dim]) quand il n'était pas possible de savoir ce qu'étaient les opérandes pendant la conversion d'un code Matlab vers Scilab. Cette fonction va déterminer la sémantique correcte pendant l'exécution. Pour obtenir un code plus performant on peut remplacer les appels à mtlb_diag :

  • Si x n'est pas une matrice de chaînes de caractères y = mtlb_diag(x,dim) peut être remplacé par y = diag(x[,dim])

    • Attention : mtlb_diag ne doit pas être utilisée pour une programmation manuelle.
