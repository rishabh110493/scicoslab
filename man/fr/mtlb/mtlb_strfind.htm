mtlb_strfind - Fonction d'émulation de la fonction Matlab strfind

Description

Dans plusieurs cas particuliers, la fonction Matlab strfind et la fonction Scilab strindex se comportent différemment dans Matlab et dans Scilab :

  • Avec des paramètres d'entrée n'étant pas des chaînes de caractère : La fonction Matlab strfind peut être utilisée avec des entrées n'étant pas des chaînes de caractères mais pas la fonction Scilab strindex.

    • La fonction mtlb_strfind(A,B) est utilisée par mfile2sci pour remplacer strfind(A,B) quand il n'était pas possible de savoir ce qu'étaient les paramètres d'entrée pendant la conversion d'un code Matlab vers Scilab. Cette fonction va déterminer la sémantique correcte pendant l'exécution. Pour obtenir un code plus performant on peut remplacer les appels à mtlb_strfind :

  • Si A et B sont des chaînes de caractères mtlb_strfind(A,B) peut être remplacé par strindex(A,B)

    • Attention : mtlb_strfind ne doit pas être utilisée pour une programmation manuelle.
