mtlb_logical - Fonction d'émulation de la fonction Matlab logical

Description

Il n'y a pas de fonction Scilab équivalent à la fonction Matlab logical mais sont comportement peut être reproduit.

La fonction mtlb_logical(A) est utilisée par mfile2sci pour remplacer logical(A) quand il n'était pas possible de savoir ce qu'était le paramètre d'entrée pendant la conversion d'un code Matlab vers Scilab. Cette fonction va déterminer la sémantique correcte pendant l'exécution. Pour obtenir un code plus performant on peut remplacer les appels à mtlb_logical :

  • Si A est une matrice de booléens mtlb_logical(A) peut être remplacé par A
  • Si A n'est pas une matrice vide mtlb_logical(A) peut être remplacé par A<>[]
  • Si A est une matrice vide mtlb_logical(A) peut être remplacé par []

    • Attention : mtlb_logical ne doit pas être utilisée pour une programmation manuelle.
