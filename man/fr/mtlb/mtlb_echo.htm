mtlb_echo - Fonction d'émulation de la fonction Matlab echo

Description

Il n'y a pas d'équivalent Scilab pour la fonction Matlab echo mais quelques cas particuliers peuvent être remplacés par des appels à la fonction Scilab mode:

  • echo : est équivalent à mode(abs(mode()-1)) dans Scilab pour les scripts et les fonctions non-compilées
  • echo on : est équivalent à mode(1) dans Scilab pour les scripts et les fonctions non-compilées
  • echo off : est équivalent à mode(0)

    • La fonction mtlb_echo(arg1[,arg2]) est utilisée par mfile2sci pour remplacer echo arg1 [arg2] quand il n'était pas possible de savoir ce qu'étaient les paramètres d'entrée pendant la conversion d'un code Matlab vers Scilab. Cette fonction va déterminer la sémantique correcte pendant l'exécution. Pour obtenir un code plus performant on peut remplacer les appels à mtlb_echo :

  • Si arg1 est égal à "on" mtlb_echo(arg1) peut être remplacé par mode(1)
  • Si arg1 est égal à "off" mtlb_echo(arg1) peut être remplacé par mode(0)

    • Attention : mtlb_echo ne doit pas être utilisée pour une programmation manuelle.
