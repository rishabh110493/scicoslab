mtlb_zeros - Fonction d'émulation de la fonction Matlab zeros

Description

Dans plusieurs cas particuliers, la fonction zeros se comporte différemment dans Matlab et dans Scilab :

  • Avec une entrée qui est un scalaire : La fonction Matlab zeros retourne une matrice n x n alors que Scilab retourne 0.

    • La fonction mtlb_zeros(A) est utilisée par mfile2sci pour remplacer zeros(A) quand il n'était pas possible de savoir ce qu'étaient les opérandes pendant la conversion d'un code Matlab vers Scilab. Cette fonction va déterminer la sémantique correcte pendant l'exécution. Pour obtenir un code plus performant on peut remplacer les appels à mtlb_zeros :

  • Si A est un scalaire mtlb_zeros(A) peut être remplacé par zeros(A,A)
  • Si A n'est pas un scalaire mtlb_zeros(A) peut être remplacé par zeros(A)

    • Attention : mtlb_zeros ne doit pas être utilisée pour une programmation manuelle.
