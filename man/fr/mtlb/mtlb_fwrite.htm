mtlb_write - Fonction d'émulation de la fonction Matlab write

Description

Il n'y a pas d'équivalent Scilab pour la fonction Matlab write. La fonction Scilab mput et la fonction Matlab write se comportent différemment dans beaucoup de cas mais sont équivalentes dans plusieurs cas.

La fonction mtlb_write(varargin) est utilisée par mfile2sci pour remplacer write. Cette fonction va déterminer la sémantique correcte pendant l'exécution. Il est quelquefois possible de remplacer les appels à mtlb_write par des appels à mput.

Attention : mtlb_write ne doit pas être utilisée pour une programmation manuelle.
