mtlb_linspace - Fonction d'émulation de la fonction Matlab linspace

Description

Dans plusieurs cas particuliers, la fonction linspace se comporte différemment dans Matlab et dans Scilab :

  • Avec les chaînes de caractère: La fonction Matlab linspace(A,B[,n]) renvoie un vecteur de caractères si A ou B est une chaîne de caractère, mais la fonction Scilab ne marche pas avec des entrées de ce type.

    • La fonction mtlb_linspace(A,B[,n]) est utilisée par mfile2sci pour remplacer linspace(A,B[,n]) quand il n'était pas possible de savoir ce qu'étaient les paramètres d'entrée pendant la conversion d'un code Matlab vers Scilab. Cette fonction va déterminer la sémantique correcte pendant l'exécution. Pour obtenir un code plus performant on peut remplacer les appels à mtlb_linspace :

  • Si A et B ne sont pas des chaînes de caractères, mtlb_linspace(A,B[,n]) peut être remplacé par linspace(A,B[,n])
  • Si A ou B est une chaîne de caractères, mtlb_linspace(A,B[,n]) peut être remplacé par ascii(linspace(ascii(A),ascii(B)[,n]))

    • Attention : mtlb_linspace ne doit pas être utilisée pour une programmation manuelle.
