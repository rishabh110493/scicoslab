mtlb_is - Fonction d'émulation de l'insertion dans une chaîne de caractères de Matlab

Description

L'insertion se comporte différemment dans Matlab et dans Scilab pour les chaînes de caractères étant donné que celles-ci ne sont pas considérées de la même façon.

La fonction str = mtlb_is(sto,sfrom,i,j) est utilisée par mfile2sci pour remplacer les opérations d'insertion dans des chaînes de caractères. Cette fonction va déterminer la sémantique correcte pendant l'exécution. Elle ne peut être remplacée.

Attention : mtlb_is ne doit pas être utilisée pour une programmation manuelle.
