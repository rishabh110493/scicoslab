mtlb_box - Fonction d'émulation de la fonction Matlab box

Description

Il n'y a pas de fonction Scilab équivalente à la fonction Matlab box mais elle peut facilement être remplacée.

La fonction mtlb_box([axes_handle[,val]]) est utilisée par mfile2sci pour remplacer box([axes_handle[,val]]) quand il n'était pas possible de savoir ce qu'étaient les paramètres d'entrée pendant la conversion d'un code Matlab vers Scilab. Cette fonction va déterminer la sémantique correcte pendant l'exécution. Pour obtenir un code plus performant on peut remplacer les appels à mtlb_box :

  • Pour un appel sans paramètre d'entrée, mtlb_box() peut être remplacé par a=gca();if a.box=="on" then a.box="off";else a.box="on";end;
  • Si axes_handle est une chaîne de caractères, mtlb_box(axes_handle) peut être remplacé par a=gca();a.box=convstr(axes_handle,"l");
  • Si axes_handle est un handle graphique, mtlb_box(axes_handle) peut être remplacé par if axes_handle.box=="on" then axes_handle.box="off";else axes_handle.box="on";end;
  • Si axes_handle est un handle graphique et val est une chaîne de caractères, mtlb_box(axes_handle,val) peut être remplacé par axes_handle.box=convstr(val,"l");

    • Attention : mtlb_box ne doit pas être utilisée pour une programmation manuelle.
