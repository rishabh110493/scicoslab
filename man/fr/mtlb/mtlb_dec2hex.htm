mtlb_dec2hex - Fonction d'émulation de la fonction Matlab dec2hex

Description

Dans plusieurs cas particuliers, la fonction dec2hex se comporte différemment dans Matlab et dans Scilab :

  • Avec la matrice vide : La fonction Matlab dec2hex retourne "" mais la fonction Scilab renvoie [].
  • Avec des paramètres d'entrée complexes : La fonction Matlab dec2hex enlève automatiquement la partie imaginaire mais pas la fonction Scilab.
  • La fonction Matlab dec2hex renvoie toujours un vecteur colonne mais la fonction Scilab renvoie un matrice de la même taille que le paramètre d'entrée.
  • La fonction Matlab dec2hex peut avoir deux paramètres d'entrée mais pas dans Scilab.

    • La fonction mtlb_dec2hex(D[,N]) est utilisée par mfile2sci pour remplacer dec2hex(D[,N]) quand il n'était pas possible de savoir ce qu'étaient les paramètres d'entrée pendant la conversion d'un code Matlab vers Scilab. Cette fonction va déterminer la sémantique correcte pendant l'exécution. Pour obtenir un code plus performant on peut remplacer les appels à mtlb_dec2hex :

  • Si D n'est pas une matrice vide, mtlb_dec2hex(D) peut être remplacé par matrix(dec2hex(real(D)),-1,1) si D est complexe et par matrix(dec2hex(D),-1,1) dans les autres cas.

    • Attention : mtlb_dec2hex ne doit pas être utilisée pour une programmation manuelle.
