mtlb_findstr - Fonction d'émulation de la fonction Matlab findstr

Description

Il n'y a pas de fonction Scilab équivalente à la fonction Matlab findstr.

La fonction mtlb_findstr(A,B) est utilisée par mfile2sci pour remplacer findstr(A,B) quand il n'était pas possible de savoir ce qu'étaient les paramètres d'entrée pendant la conversion d'un code Matlab vers Scilab. Cette fonction va déterminer la sémantique correcte pendant l'exécution. Pour obtenir un code plus performant on peut remplacer les appels à mtlb_findstr :

  • Si A est toujours plus longue que B (i.e. strfind peut remplacer findstr en Matlab), mtlb_findstr(A,B) peut être remplacé par strindex(A,B)

    • Attention : mtlb_findstr ne doit pas être utilisée pour une programmation manuelle.
