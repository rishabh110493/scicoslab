Scilab Function
Last update : 29/02/2016

startup - startup file

Description

Les fichiers de démarrage SCIHOME/.scilab et SCIHOME/scilab.ini sont automatiquement executés (s'ils existent) au démarrage de ScicosLab, ainsi que le fichier SCI/scilab.star présent dans le répertoire de ScicosLab.

Remarks

les fichiers de démarrage doivent obligatoirement comporter une nouvelle ligne en fin de fichier pour être pris en compte.

Définition de SCIHOME : Sous Windows : C:/Documents and Settings/<User>/ScicosLab/<ScicosLab-Version> Sous Linux/Unix : /home/<User>/.ScicosLab/<ScicosLab-Version>
