beep - Produit le son "beep"

Calling Sequence

beep();
beep('on')
beep('off')
s=beep()

Description

beep() produit le son "beep".

beep('on') active le "beep"

beep('off') desactive le "beep"

s=beep() retourne le mode courantreturns the current beep mode (on or off).
