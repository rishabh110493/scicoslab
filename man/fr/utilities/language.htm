Last update : 29/02/2016

LANGUAGE - Variable qui définit la langue

Description

La variable globale LANGUAGE est une chaîne de caractères. Elle définit la langue pour l'aide en ligne et divers menus. Actuellement, les valeurs de LANGUAGE peuvent être "fr" pour le français, c'est le défaut, ou "eng" pour l'anglais.

Cette variable est définie dans le fichier d'initialisation scilab.star. On peut la changer là.

Quand on lance Scilab, on peut donner la valeur de la langue comme argument : scilab -l <language> pour Unix et runscilab.exe -l <language> pour Windows.
