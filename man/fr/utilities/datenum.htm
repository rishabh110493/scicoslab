datenum - Convertie une date en un nombre

Calling Sequence

N = datenum()
N = datenum(DT)
N = datenum(Y, M, D)
N = datenum(Y, M, D, H, MI, S)

Description

La fonction datenum convertie une date en un nombre lui correspondant. Ce nombre correspond au nombre de jours passe depuis une date reference. Par defaut, cette date est le 1-Jan-0000.

N = datenum() retourne le nombre correspondant a la date courante.

N = datenum(Y, M, D) retourne le nombre correspondant aux elements Y, M, D (annee, mois, jour).

N = datenum(Y, M, D, H, MI, S) retourne le nombre correspondant aux elements Y, M, D, H, MI, et S (annee, mois, jour, heure, minute, et seconde).

Examples

datenum()

datenum(1973,8,4)
