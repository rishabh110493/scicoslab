winopen - Ouvre le fichier avec l'application appropriée (Windows seulement)

Calling Sequence

winopen('filename')

Description

winopen('filename') ouvre le fichier avec l'application appropriée de Microsoft Windows. winopen effectue la même action que si vous double-cliquez sur l'icone dans l'explorateur de Windows.
