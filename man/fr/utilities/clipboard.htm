clipboard - Copie ou colle une chaine de caracteres dans le presse-papier (Windows).

Calling Sequence

clipboard('copy',data)
str=clipboard('paste')
clipboard('do','paste')
clipboard('do','copy')
clipboard('do','empty')
clipboard(winnum,'EMF')
clipboard(winnum,'DIB')

Description

clipboard('copy',data) copie le contenu de data dans le presse-papier. Si data n'est pas une chaine de caracteres, sci2exp est utilisé pour convertir data en chaine de caracteres.

str = clipboard('paste') retourne le contenu du presse-papier dans la variable str, ou [] si le contenu du presse-papier ne peut etre converti en chaine de caracteres.

clipboard('do','paste') , clipboard('do','copy') , clipboard('do','empty') effectue l'action equivalente au menu Coller, Copier et Vider le presse-papier.

Note : clipboard(...) ne fonctionne que pour Windows et qu'en mode interface graphique.

clipboard(winnum,'EMF') copie, au format EMF, dans le presse papier le graphe correspondant au numero de la fenetre associée.

clipboard(winnum,'DIB')copie, au format DIB, dans le presse papier le graphe correspondant au numero de la fenetre associée.
