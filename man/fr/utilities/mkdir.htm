mkdir - Crée un nouveau répertoire

Calling Sequence

mkdir('dirname')
mkdir('parentdir','newdir')
status=mkdir( ... )
[status,msg]=mkdir( ... )

Description

mkdir('dirname') crée le repertoire dirname dans le répertoire courant.

mkdir('parentdir','dirname') crée le répertoire dirname dans le répertoire existant parentdir, ou parentdir est un chemin relatif ou absolu.

[status,message] = mkdir(...,'dirname') crée le répertoire dirname dans le répertoire existant parentdir, retourne le status, et un message. Ici, status vaut 1 si tout c'est bien passé, 2 si le repertoire existe déjà, 0 dans un autre cas.

Examples

rmdir(SCI + '/Directory'); //détruit le répertoire s'il existe
mkdir(SCI,'Directory')
status=mkdir(SCI,'Directory')
[status,msg] = mkdir(SCI,'Directory')
rmdir(SCI+'/Directory')
