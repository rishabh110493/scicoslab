obsv_mat - observability matrix

Calling Sequence

[O]=obsv_mat(A,C)
[O]=obsv_mat(sl)

Parameters

  • A,C,O: real matrices
  • sl: syslin list

Description

obsv_mat returns the observability matrix:


O=[C; CA; CA^2;...; CA^(n-1) ]
