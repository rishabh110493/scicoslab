Maxplus Function
Last update : 29/02/2016

star - maxplus star

Calling Sequence

B=star(A)

Parameters

  • A : maxplus full square matrix
  • B : Id+A+A^2+.....

Description

Solve x=Ax+Id in the maxplus algebra. When there is no circuits with positive weight in G(A) (the incidence graph of A) B=Id+A+.....A^(n-1) where n denotes the order of the square matrix A.

Examples

star(#(1))
star(#(-1))
star(#([1,2;3,4]))
a=#([-3,-2;-1,0]); b=star(a)
b==%eye(2,2)+a
b==b+a*a
