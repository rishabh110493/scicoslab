equalspan - maxplus matrix image equality

Calling Sequence

b=equalspan(A,B,[pre])

Parameters

  • A : full maxplus matrix
  • B : full maxplus matrix
  • pre : precision
  • b : boolean

Description

b is true if Im(B)=Im(A) else it is false. The equality is equal up to pre which is 0 by default.

Examples

a=#([1,0;0,1]);
equalspan(a,a)
equalspan(a,a*10)
equalspan(a,#([1,1;2,0]))
