Maxplus Function
Last update : 29/02/2016

plus - (+) maxplus matrix plus operator

Calling Sequence

B=plus(A)

Parameters

  • A : maxplus full square matrix
  • B : A+A^2+.....

Description

Compute A*A^*=A+A^2+..... of a maxplus matrix A.

Examples

plus(#(1))
plus(#(-1))
plus(#([1,2;3,4]))
a=#([-3,-2;-1,0]); b=plus(a); b==a*star(a)
