Maxplus Operator
Last update : 29/02/2016

minus - (-) maxplus elementwise minus operator

Calling Sequence

A-B

Parameters

  • A : maxplus scalar or full maxplus matrix
  • B : maxplus scalar or full maxplus matrix

Description

Minus.

Elementwise minus of nonzero entries of two maxplus matrices (a-%0=a).

Examples

a=#(1);b=#(2); a-b
3-%0
%0-%0
a=#([1,2;3,4]); b=#([1,1;%0,1]); a-b
