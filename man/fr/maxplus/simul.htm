simul - maxplus linear system simulation

Calling Sequence

y=simul(S,u)

Parameters

  • S : maxplus linear system
  • u : input
  • y : output

Description

Computes the output y of a maxplus linear system S when the input is u.

Examples

S=mpsyslin([%0,1;10,%0],[%1;%1],[%1,%1])
simul(S,[1:10])
