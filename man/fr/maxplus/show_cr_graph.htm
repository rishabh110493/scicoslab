show_cr_graph - flowshop critical graph (maxplus)

Calling Sequence

gh=show_cr_graph(g)

Parameters

  • g : flowshop graph built with flowshop_graph
  • gh : graph of the flowshop with critical graph highlighted

Description

Shows the critical graph (bottleneck) of a flowshop described by g in a metanet window.

Examples

PT=#([2,1;%0,1;4,%0]);
xsetech([0 0 0.5 0.5]);
[g,T,N]=flowshop_graph(PT,[1,1,1],[2,2],50);
show_cr_graph(g);
