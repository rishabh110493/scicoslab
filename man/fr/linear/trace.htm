trace - trace d'une matrice

Calling Sequence

trace(X)

Parameters

  • X : matrice carrée, réelle, complexe, polynomiale ou rationnelle.

Description

trace(X) calcule la trace de X.

Identique à sum(diag(X)).

Examples

A=rand(3,3);
trace(A)-sum(spec(A))
