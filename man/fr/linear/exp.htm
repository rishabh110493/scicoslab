exp - exponentielle

Calling Sequence

exp(X)

Parameters

  • X : matrice carrée réelle ou complexe.

Description

exp(X) renvoie (élément par élément) l'exponentielle des éléments de X.

Examples

x=[1,2,3+%i];
log(exp(x))  // élément par élément
2^x
exp(x*log(2))
