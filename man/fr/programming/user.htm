user - interfaçage d'une routine Fortran ou C

Calling Sequence

[s_1,s_2,...,s_lhs]=user(e_1,e_2,...,e_rhs)

Description

Avec cette commande il est possible d'utiliser un programme externe comme une commande Scilab où (s_1,s_2,...,s_lhs) sont les variables de sortie et (e_1,e_2,...,e_rhs) sont les variables d'entrée. Pour utiliser cette commande dans Scilab il faut écrire quelques lignes dans la subroutine Fortran user de Scilab. Voir intersci ou la documentation de Scilab pour plus d'informations.
