Scilab Function
Last update : 29/02/2016

sciargs - arguments sur la ligne de commande ScicosLab

Calling Sequence

args=sciargs()

Description

Cette fonction renvoie un vecteur de chaînes de caractères contenant les arguments de la commande de lancement de ScicosLab. Le premier terme du vecteur args contient le nom complet (avec le chemin) du fichier exécutable de Scilab.

Cette fonction correspond à la fonction getarg en langage C.
