debug - niveau de déboggage

Calling Sequence

debug(level-int)
level-int=debug()

Parameters

  • level-int : entier (entre 0 et 4)

Description

Pour les valeurs 0,1,2,3,4 de level-int , debug définit différents niveaux de déboggage (pour les experts de Scilab !).
