whereis - nom de la librairie contenant une fonction

Calling Sequence

[librname]=whereis(function-name)

Description

Renvoie une chaîne de caractères contenant le nom de la librairie contenant la fonction function-name. Le chemin d'accès à la librairie peut être obtenu avec la fonction "librname".
