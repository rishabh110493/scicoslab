empty ([]) - matrice vide

Description

[] désigne une matrice vide. Elle est définie de façon unique et possède 0 lignes et 0 colonnes, soit size([]) =[0,0] . Les convention suivantes sont adoptées :

[] * A = A * [] = []

[] + A = A + [] = A

[ [], A] = [A, []] = A inv([]) =[]

det([])=cond([])=rcond([])=1, rank([])=0

Les fonctions matricielles renvoient [] ou un message d'erreur quand il n'y pas de réponse évidente. Les systèmes dynamiques linéaires (listes de type syslin) peuvent avoir plusieurs lignes ou colonnes.

Examples

s=poly(0,'s'); A = [s, s+1]; 
A+[], A*[]
A=rand(2,2); AA=A([],1), size(AA)
svd([])
w=ssrand(2,2,2); wr=[]*w; size(wr), w1=ss2tf(wr), size(w1)
