Description

Une sequence de deux slash consécutifs // en dehors de la définition d'une chaîne de caractères marque le début d'un commentaire. Les deux slash ainsi que les caractères suivant jusqu'à la fin de la ligne ne sont pas interprétées par Scilab.

Dans la définition d'une fonction, les premières lignes de commentaires (jusqu'à la première instruction ou la première ligne vide peuvent fournir les éléménts du help par défaut de la fonction.

Examples


	g=9.81// la gravité
	
	text='a//b'
	
	function y=myfunction(x)
	// myfunction calcule y=x^2+1
	// x doit être un vectuer ou une matrice
	y=x^2+1
	endfunction
	
	help myfunction
