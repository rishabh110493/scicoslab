date - date courante sous forme d'une chaîne de caractères

Calling Sequence

dt=date()

Parameters

  • dt : chaîne de caractères

Description

dt=date() renvoie une chaîne de caractères contenant la date au format jj-mmm-aaaaa.

Examples

date()
