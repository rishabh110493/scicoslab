semicolon (;) - séparateur de lignes ou d'instructions Scilab

Description

Les points-virgules sont utilisés pour séparer des lignes dans une définition de matrice.

Les points-virgules sont utilisés aussi pour marquer la fin d'une instruction. Dans ce cas le résultat n'est pas affiché. On peut utiliser à la place une virgule (,) qui permet de séparer les instructions sans supprimer l'affichage.

Examples

a=[1,2,3;4,5,6];
a=1;b=1,c=2
