Palette

  • Sinks - Palette Affichage

Description

That block is an actuator. It can be used with the Scicos Code Generation.

Boîte de dialogue

  • Input Size
    Définit la taille de la matrice du port d'entrée régulier.
    Propriétés : Type 'mat' de taille [1,2].

  • Input Type
    Définit le type du port d'entrée régulier.
    Propriétés : Type 'vec' de taille 1.

  • Accept herited events 0/1
    Options pour choisir si le bloc hérite d'évenements par son port d'entrée régulier ou bien explicitement par un port d'entrée événementiel (0).
    Propriétés : Type 'vec' de taille 1.

  • Actuator number
    Un entier qui définit le numéro de port.
    Propriétés : Type 'vec' de taille 1.

Propriétés par défaut

  • toujours actif: non
  • direct-feedthrough: oui
  • détection de passage à zéro: non
  • mode: non
  • entrée régulières:
    - port 1 : taille [1,1] / type 1
  • nombre des entrées évènementielles: 1
  • nombre des sorties évènementielles: 0
  • possède un état continu: non
  • possède un état discret: non
  • possède un état objet: non
  • nom de la fonction de calcul: bidon
