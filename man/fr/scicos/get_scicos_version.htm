get_scicos_version - Retourne la version courante de Scicos

Module

  • Scicos

Description

Cette fonction retourne le numéro de version courant de Scicos.

Séquence d'appel 

scicos_ver = get_scicos_version

Paramètres

  • scicos_ver : une chaîne de caractère indiquant le numéro de version.

Exemple 

//Retourne la version de scicos 
get_scicos_version()

Contenu du fichier
