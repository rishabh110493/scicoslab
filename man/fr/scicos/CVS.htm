Palette

  • Electrical.cosf - Boîte à outils des composants électriques

Description

This block is an ideal voltage source. The voltage value is controlled through the explicit input of the block (connected to standard Scicos blocks). The current passing through the block is independent of the voltage across the block terminals.

Propriétés par défaut

  • Entrées :
    • Nom de la variable Modelica : 'vin'
      Variable explicite.

    • Nom de la variable Modelica : 'p'
      Variable implicite.

  • Sorties :
    • Nom de la variable Modelica : 'n'
      Variable implicite.

  • Nom du fichier model : CVS
