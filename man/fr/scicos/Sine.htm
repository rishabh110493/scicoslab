Palette

  • ModLinear.cosf - Boîte à outils des composants linéaires

Description

This block is a sine wave Modelica block signal generator.

Boîte de dialogue

  • amplitude
    Amplitude of sine wave.
    Properties : Type 'vec' of size 1.

  • freqHz
    Frequency of sine wave.
    Properties : Type 'vec' of size 1.

  • phase
    Phase of sine wave.
    Properties : Type 'vec' of size 1.

  • offset
    Offset of output signal.
    Properties : Type 'vec' of size 1.

  • startTime
    Output = offset for time lower than or equal to startTime.
    Properties : Type 'vec' of size 1.

Propriétés par défaut

  • Sorties :
    • Nom de la variable Modelica : 'RealOutputx'
      Variable implicite.

  • Paramètres :
    • Nom du paramètre Modelica : 'amplitude'
      Valeur par défaut : 1
      Variable d'état : non.

    • Nom du paramètre Modelica : 'freqHz'
      Valeur par défaut : 1
      Variable d'état : non.

    • Nom du paramètre Modelica : 'phase'
      Valeur par défaut : 0
      Variable d'état : non.

    • Nom du paramètre Modelica : 'offset'
      Valeur par défaut : 0
      Variable d'état : non.

    • Nom du paramètre Modelica : 'startTime'
      Valeur par défaut : 0
      Variable d'état : non.

  • Nom du fichier model : Sine
