Palette Scicos
fr - eng


Palette Affichage

Module

  • Scicos

Description

Dans la palette Sinks, vous pouvez trouver une variété de blocs utilisés pour afficher (Scope) et écrire les données dans des fichiers pendant la simulation ainsi que des ports de sortie utilisés dans les superblocs.
Les blocs de cette palette ne possèdent pas de ports réguliers de sortie.

Blocs

  • ACTUATOR_f - Actionneur
  • AFFICH_m - Affichage
  • BUSOUT_f - Port de sortie BUS
  • CANIMXY - Scope animé y=f(x)
  • CANIMXY3D - Scope animé z=f(x,y)
  • CEVENTSCOPE - Scope pour événements
  • CFSCOPE - Scope flottant
  • CLKOUTV_f - Port de sortie d'activation
  • CMAT3D - Scope matriciel 3D
  • CMATVIEW - Scope matriciel coloré
  • CMSCOPE - Scope à affichage multiple
  • CSCOPE - Scope
  • CSCOPXY - Scope y=f(x)
  • CSCOPXY3D - Scope z=f(x,y)
  • OUT_f - Port de sortie régulier
  • OUTIMPL_f - Port implicite de sortie
  • TOWS_c - Données vers l'environnement Scilab
  • WFILE_f - Ecriture dans un fichier
  • WRITEAU_f - Ecrite de fichier son AU
  • WRITEC_f - Ecriture de donnée binaire
