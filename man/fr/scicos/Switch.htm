Palette

  • Electrical.cosf - Boîte à outils des composants électriques

Description

This is a non-ideal two-pole switch. If the explicit input become positive, two pins are connected via a resistor of resistance RON). Otherwise, two pins are connected via ROFF resistance. Note that using this block may result in a stiff model, so try to choose proper error tolerances.

Boîte de dialogue

  • Resistance in On state (Ohm)
    Switch resistance when the Switch is closed
    Propriétés : Type 'vec' de taille 1.

  • Resistance in Off state (Ohm)
    Switch resistance when the switch is open
    Propriétés : Type 'vec' de taille 1.

Propriétés par défaut

  • Entrées :
    • Nom de la variable Modelica : 'p'
      Variable implicite.

    • Nom de la variable Modelica : 'inp'
      Variable explicite.

  • Sorties :
    • Nom de la variable Modelica : 'n'
      Variable implicite.

  • Paramètres :
    • Nom du paramètre Modelica : 'Ron'
      Valeur par défaut : 0.01
      Variable d'état : non.

    • Nom du paramètre Modelica : 'Roff'
      Valeur par défaut : 100000
      Variable d'état : non.

  • Nom du fichier model : Switch
