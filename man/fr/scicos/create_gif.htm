create_gif - Crée des icones gif de blocs Scicos

Module

  • Scicos

Description

Crée des icones au format gif de blocs Scicos. La liste des noms des blocs peut être donnée via TXT. Le paramètre path contient le chemin cible où les fichiers sont crées. La couleur de fond est transparente par l'utilisation de la commande "convert" (si disponible). Si la fonction est appelée sans arguments alors les icones des blocs des librairies standards de Scicos sont créees et placées dans le répertoire SCI+'/macros/scicos/scicos_doc/man/gif_icons'.

Séquence d'appel 

create_gif([TXT,path])

Paramètres

  • TXT : un vecteur de chaînes de caractères donnant la liste des fichiers de fonctions d'interfacages de bloc Scicos.
  • path : le chemin cible où sont enregistrés les icones gif.

Contenu du fichier

  • SCI/macros/util/create_gif.sci [voir code]
