model2blk - Crée une structure scicos_block à partir d'un modèle Scicos

Module

  • Scicos

Description

Cette fonction est utilisée par obtenir une structure de simulation scicos block à partir d'une structure de donnée scicos_model d'un bloc Scicos. Le modèle du bloc doit-être correctement renseigné avant d'appeller model2blk.
Cette fonction peut être utilisée avant l'appel à la fonction de calcul via callblk.

Séquence d'appel 

[blk] = model2blk(model)

Paramètres

  • blk : une structure scicos_block
  • model : une structure scicos_model
