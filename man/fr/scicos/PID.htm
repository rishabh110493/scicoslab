Palette

  • OldBlocks - Palette des anciens blocs

Description

Ce bloc implémente un régulateur PID. Un régulateur Proportionnel Intégral Dérivé (PID) est un organe de contrôle permettant d'effectuer une régulation en boucle fermée d'un système industriel. C'est le régulateur le plus utilisé dans l'industrie et permet de contrôler la grande majorité des procédés.

Boîte de dialogue

  • Proportional
    La valeur du gain qui multiplie l'erreur.
    Propriétés : Type 'vec' de taille -1.

  • Integral
    La valeur de l'intervalle de temps sur lequel l'erreur est intégrée.(1/Integral)
    Propriétés : Type 'vec' de taille -1.

  • Derivation
    La valeur de temps suivant laquelle l'erreur est dérivée.
    Propriétés : Type 'vec' de taille -1.

Propriétés par défaut

  • toujours actif: non
  • direct-feedthrough: non
  • détection de passage à zéro: non
  • mode: non
  • entrée régulières:
    - port 1 : taille [-1,-2] / type 1
  • sorties régulières:
    - port 1 : taille [-1,-2] / type 1
  • nombre des entrées évènementielles: 0
  • nombre des sorties évènementielles: 0
  • possède un état continu: non
  • possède un état discret: non
  • possède un état objet: non
  • nom de la fonction de calcul: csuper
