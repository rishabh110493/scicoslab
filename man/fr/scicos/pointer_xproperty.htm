pointer_xproperty - Retourne le type d'une variable d'état à temps continu

Module

  • Scicos

Description

Cette fonction retourne un vecteur indiquant le type (algébrique ou différentiel) des variables d'états continues utilisées dans un bloc.

Séquence d'appel 

[xprop]=pointer_xproperty

Paramètres

  • xprop
    La valeur donne le type des états :
    • -1 : un état algébrique.
    • 1 : un état différentiel.
