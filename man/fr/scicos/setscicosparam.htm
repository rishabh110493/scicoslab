setscicosparam - Ecrit des données binaires dans un fichier de paramètres

Module

  • Scicos

Description

Write binary data in rdnom_params.dat file generated by Scicos CodeGeneration.

Séquence d'appel 

ok = setscicosparam(dt,fic)

Paramètres

  • dt : : a list that encloses the name and the values of the scicos variables to be written in the parameters file.
  • fic : : path+name of the data file.
  • ok : : a flag to say if the operation is succesfull.

Contenu du fichier
