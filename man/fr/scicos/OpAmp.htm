Palette

  • Electrical.cosf - Boîte à outils des composants électriques

Description

The ideal OpAmp is a two-port. The left port is fixed to v1=0 and i1=0 (nullator). At the right port both any voltage v2 and any current i2 are possible (norator).

Propriétés par défaut

  • Entrées :
    • Nom de la variable Modelica : 'in_p'
      Variable implicite.

    • Nom de la variable Modelica : 'in_n'
      Variable implicite.

  • Sorties :
    • Nom de la variable Modelica : 'out'
      Variable implicite.

  • Nom du fichier model : OpAmp
