add_palette - Ajout de palettes Scicos

Module

  • Scicos

Description

Cette fonction ajoute une palette à la fin de la liste des palettes intiales Scicos. Si une palette avec un même chemin existe déjà alors rien n'est fait. Si une palette avec un nom identique mais avec un chemin différent est détecté, alors la palette est ajoutée avec un suffixe x. Il faut noter que la variable scicos_pal est re-crée locallement dans la fonction et est passée à l'environnment supérieur par la commande scicos_pal=resume(scicos_pal).

Séquence d'appel 

add_palette(Title,path)

Paramètres

  • Title
    Le nom de la palette.
  • path
    Le chemin de la palette.
