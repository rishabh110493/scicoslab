data2sig - Données Scilab vers structure signal Scicos

Module

  • Scicos

Description

Met des données dans une structure signal scicos. Si A est un vecteur, alors elle sera comprise comme un scalaire qui évolue dans le temps. Si A est une matrice, alors elle sera comprise comme un scalaire qui évolue dans le temps. Si A est une hypermatrice, alors elle sera comprise comme une matrice qui évolue dans le temps.

Séquence d'appel 

B=data2sig(A[,step])

Paramètres

  • A : vecteur/matrice ou hypermatrice(à 3 dimensions) de nombres réels/complexes ou entiers.
  • step : un argument optionnel qui donne le pas temporel pour le champ de sortie "time". Si celui-ci n'est pas donné, alors le pas temporel est égal à 1.
  • B : une structure scilab qui définit un signal :
    • B.values : L'entrée A.
    • B.time : Les instants d'échantillons sous la forme d'un vecteur colonne. La taille du vecteur est donnée par l'entrée A.

Exemple 

-->a(:,:,4)=([7 8 9;10 11 12]);
-->b=data2sig(a,0.1)
b  =
values: [2x3x4 hypermat]
time: [4x1 constant]
-->b.time
ans  =
0.
0.1.
0.2.
0.3.
