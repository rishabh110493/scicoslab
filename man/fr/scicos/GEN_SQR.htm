Description

This block is a square wave generator. The output switches between the Minimum value and the maximum value periodicly. The value of the fourth parameter of this block defines the period of the signal. The period of the switching is equal to the period of the signal divided by 2.

Boîte de dialogue

  • Minimum Value
    The Lower bound of the generator.
    Properties : Type 'mat' of size [-1,-1].

  • Maximum Value
    The upper bound of the generator.
    Properties : Type 'mat' of size [-1,-1].

  • Initial Value( 1= Minimum Value 2= Maximum Value)
    When this parameter is set to 1, the initial value of the signal will be the lower bound. When it is set to 2, the initial value will be the upper bound. In else case, it generates an error.
    Properties : Type 'pol' of size -1.

  • Period (sec)
    The period of the square wave.
    Properties : Type 'pol' of size -1.

Propriétés par défaut

  • toujours actif: non
  • direct-feedthrough: non
  • détection de passage à zéro: non
  • mode: non
  • sorties régulières:
    - port 1 : taille [-1,-2] / type -1
  • nombre des entrées évènementielles: 0
  • nombre des sorties évènementielles: 0
  • possède un état continu: non
  • possède un état discret: non
  • possède un état objet: non
  • nom de la fonction de calcul: csuper
