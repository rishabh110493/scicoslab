str2code - return scilab integer codes associated with a character string

Calling Sequence

c=str2code(str)

Parameters

  • str: a character string
  • c: vector of character integer codes

Description

Return c such that c(i) is the scilab integer code of part(str,i))

Examples


str2code('Scilab')
