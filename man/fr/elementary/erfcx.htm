erfcx - fonction erreur complémentaire normalisée.

Calling Sequence

y = erfcx(x)

Parameters

  • x : vecteur réel
  • y : vecteur réel (de même taille que x)

Description

erfcx calcule la fonction erreur complémentaire normalisée.
