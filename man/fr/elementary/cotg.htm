cotg - cotangente

Calling Sequence

[t]=cotg(x)

Parameters

  • x,t : vecteur ou matrice (réelle ou complexe)

Description

Les éléments de t sont les cotangentes des éléments de x. t=cos(x)./sin(x)

Examples

x=[1,%i];
cotg(x)-cos(x)./sin(x)
