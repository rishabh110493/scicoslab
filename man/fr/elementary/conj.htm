conj - conjugaison complexe

Calling Sequence

[y]=conj(x)

Parameters

  • x,y : vecteur ou matrice (réelle ou complexe)

Description

Les éléments de conj(x) sont les complexes conjugués des éléments de x.

Examples

x=[1+%i,-%i;%i,2*%i];
conj(x)
x'-conj(x)  // x' est la transposée conjuguée de x
