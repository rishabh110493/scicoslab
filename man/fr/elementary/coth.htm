coth - cotangente hyperbolique

Calling Sequence

[t]=coth(x)

Parameters

  • x,t : vecteur ou matrice (réelle ou complexe)

Description

Les éléments de t sont les cotangentes hyperboliques des éléments de x.

Examples

x=[1,2*%i]
t=exp(x);
(t-ones(x)./t).\(t+ones(x)./t)
coth(x)
