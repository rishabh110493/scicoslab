tanh - tangente hyperbolique

Calling Sequence

t=tanh(x)

Parameters

  • x,t : vecteur ou matrice (réelle ou complexe)

Description

Les éléments de t sont les tangentes hyperboliques des éléments de x.

Examples

x=[1,%i,-1,-%i]
tanh(x)
sinh(x)./cosh(x)
