cos - cosinus

Calling Sequence

[y]=cos(x)

Parameters

  • x : vecteur ou matrice (réelle ou complexe)

Description

Pour un vecteur ou une matrice, les éléments de cos(x) sont les cosinus des éléments de x. Pour le cosinus matriciel utiliser la fonction cosm(X).

Examples

x=[0,1,%i]
acos(cos(x))
