pol2str - conversion polynôme vers chaîne de caractères

Calling Sequence

[str]=pol2str(p)

Parameters

  • p : polynôme réel
  • str : chaîne de caractères

Description

convertit le polynôme p en une chaîne de caractères (fonction utilitaire).
