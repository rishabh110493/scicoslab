xtape - détermine le processus d'enregistrement des graphiques

Calling Sequence

xtape(str,[num,rect])

Parameters

  • str : chaîne de caractères, "on", "clear", "replay" ou "replaysc".
  • num : entier.
  • rect : vecteur ligne de taille 4.

Description

xtape est utilisée pour déterminer le processus d'enregistrement des graphiques :

xtape("on") sélectionne le pilote "Rec" qui enregistre toutes les opérations graphiques.

xtape("clear",num) efface la fenêtre graphique num ainsi que les graphiques enregistrés.

xtape("replay",num) ré-exécute les ordres graphiques enregistrés dans la fenêtre num.

xtape("replaysc",num,rect) redessine le contenu de la fenêtre graphique num en utilisant rect=[xmin,ymin,xmax,ymax] comme bornes.
