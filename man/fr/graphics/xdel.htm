xdel - supprime des fenêtres graphiques

Calling Sequence

xdel([win-nums])

Parameters

  • win-nums : entier ou vecteur d'entiers

Description

xdel supprime les fenêtres graphiques dont les numéros sont donnés dans le vecteur win-nums ou supprime la fenêtre graphique courante si aucun argument n'est donné.
